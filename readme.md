A integração deverá pegar os boletos que estão para vencer nos próximos 5 dias no sistema SIGA e enviar um request para o MegaZap.

Documentação do SIGA: https://siga04.activesoft.com.br/docs/
Documentação do MegaZap: https://megazap.readme.io/reference/criarparawhatsappqrcode

No SIGA, para visualizar os boletos, é necessário passar aluno por aluno (que está ativo em 2026), e pegar a informação dos boletos a vencer (nos próximos 5 dias).
